## Role Capabilities

### Supervisor
- Create and manage requests
- Archive/unarchive requests
- View archived requests
- Reassign requests to other users
- Manage users

### Clerk
- Create and manage requests
- Upload and redact files
- Export redacted files
- Cannot archive requests
- Cannot view archived requests
